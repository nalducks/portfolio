About Me

Saya lulusan SMK Teknik Elektronika pada tahun 2021, saya mempunyai landasan yang kuat di bidang elektronika dan teknologi. Saya tertarik untuk mengejar karir di bidang sumber daya manusia.

Selain itu, saya juga mempelajari bahasa pemrograman seperti C++, Python, Java Script, dan juga HTML dan CSS. Saya ingin memulai karir saya di bidang pemrograman. Pengalaman saya dalam pemrograman termasuk membuat situs portofolio sendiri, membuat program lampu merah, rumah pintar, pengatur waktu, dan kalkulator.

Pengalaman saya di bidang elektronik mencakup melakukan perawatan dan perbaikan pada berbagai mesin, termasuk mesin las inverter, kompresor, penyedot debu, dan pembersih jet. Selain itu, saya memiliki pengalaman memecahkan masalah dan memperbaiki mesin industri, membuat sirkuit bintang-delta dan maju-mundur, serta memprogram inverter untuk mengoperasikan motor 3 fase.

Selain bidang ini, saya juga suka jalan-jalan, bermain musik, atau mempelajari hal baru.
